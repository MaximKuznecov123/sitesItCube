Что нужно знать?

Чтобы стать WEB-разработчиком, необходимо освоить следующие навыки:

  1. HTML и CSS. Знать синтаксис этих языков. Уметь с их помощью воспроизводить внешний вид, созданный дизайнером. Понимать, как создавать интерактивные элементы.
  2. Интерфейс среды для работы. Писать HTML-код можно и просто в блокноте. Однако удобнее делать это в специальных инструментах — редакторах кода, например Visual Studio Code или Codeanywhere.
  3. Адаптивная вёрстка. Уметь с помощью инструментов HTML и CSS подстраивать страницы под размер экранов. Знать принципы вёрстки для мобильных устройств.
  4. Основы дизайна. Понимать сочетания шрифтов и цветов
  5. CMS и конструкторы. Знание популярных систем управления сайтами: WordPress, Bitrix, Tilda. Умение загрузить туда написанный код страницы, создать или адаптировать шаблон.
  6. JavaScript. Уметь написать на языке программирования простую анимацию
