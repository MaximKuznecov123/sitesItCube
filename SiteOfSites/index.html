Ч е м у м ы

HTML

— это стандартный язык, который используется для создания веб-страниц. HTML — это скелет этой книги, костяк, который определяет структуру: где будет заголовок, абзацы, изображения и так далее

CSS

Язык таблиц стилей, который позволяет прикреплять стиль (например, шрифты и цвет). Обычно CSS-стили используются для создания и изменения стиля элементов веб-страниц

JavaScript

С точки зрения веб-разработки, без знаний этой технологии невозможно заниматься созданием современных интерактивных сайтов. Язык JS - это то, что «оживляет» разметку страниц (HTML) и пользовательский функционал (CMS) сайтов.

Основной фонд музея состоит из диорам важнейших сражений отечественной истории, включает стендовые модели колесной, артиллерийской, гусеничной и авиационной техники. Кроме того, в основных фондах музея хранится значительное количество экспонатов, обнаруженных на полях сражений в Смоленской, Новгородской, Воронежской и Волгоградской области по результатам экспедиций поискового отряда "Рубеж" г. Балашова. В основном фонде музея 227 единиц хранения, вспомогательный фонд насчитывает 43 единицы хранения.
